# evlm

A package to evaluate Vision-Lanague Models

## Modesl Suported: 
### Base Enviorment: 

#### CLIP Style VLMs:
* ALIGN      (General)
* BLIP       (General)
* BLIP2      (General)
* OpenCLIP   (General)
* OwlVIT     (General)
* OwlVIT2    (General)
* BiomedCLIP (Biomedical)
* PLIP       (Pathology)
* Quilt      (Pathology)

#### Generative Style VLMs:
* InstructBlip   (General)
* CogVLM         (General)
* QwenChatModel  (General)


### Conch Enviorment: 
* CONCH      (Pathology)

## Installation

